## Round 3 Work Plan Reflection & Module Integration Reflextion
Our prototype is designed and mostly ready to be printed, however, we are discussing changing aspects of the design and so are taking our time to print. We are still debating on if we want a front module to gather the dinosaurs, or if that will delay us in completing the competition and just add more weight to the robot. we have so far finalized the barebones structure of the robot - base plate, manipulator, tyres and axle. They work well and fit together, but in an improv print, we added a hook to attach an elastic for added torque when trying to lift props. We need to do more testing with the chassis to make sure battery and electronic components fit well.

We believe the workload is more hardware focused right now and will get more software focused as we integrate modules, so Ariya is contributing to the robot mroe righ tnow and Samuel is organising his code files and discussing on how to approach it to fit the team's needs.

The project is within the specified budget and we have all resources available. Most of the delays are because of the wait times for the 3D printing and laser cutting.
